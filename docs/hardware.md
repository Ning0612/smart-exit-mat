# 硬體需求與接線

## 元件清單

| 元件 | 規格 / 型號 | 數量 | 說明 |
|------|------------|------|------|
| 微控制器 | NodeMCU-32S（ESP32） | 1 | 主控板，內建 4 MB Flash、Wi-Fi、藍牙 |
| 重量感測模組 | HX711 ADC 模組 | 1 | 24-bit ADC，專為壓力感測設計 |
| 應變計 | 50 kg Load Cell（應變計） | 4 | 惠斯通電橋式感測器 |
| 地墊 | 任意尺寸硬底地墊 | 1 | 需可固定四顆感測器於四角 |
| 電源 | USB 5V / 1A | 1 | 透過 Micro-USB 或 USB-C 供電 |
| 連接線 | 杜邦線（公/母） | 若干 | HX711 與 ESP32 間的訊號連接 |
| 螺絲 / 固定座 | M4 或 M5 | 各 4 | 固定感測器於地墊底板四角 |

> **電流需求**：NodeMCU-32S 在 WiFi 活躍時峰值約 240 mA，HX711 約 1.5 mA，合計不超過 300 mA。一般手機充電頭（5V/1A）即可穩定供電。

## HX711 → NodeMCU-32S 接線

### 訊號線對應

| HX711 腳位 | NodeMCU-32S GPIO | 說明 |
|-----------|-----------------|------|
| VCC | 3.3V | 感測器電源（不可接 5V，HX711 工作電壓 2.7–5.5 V，但 3.3V 可用） |
| GND | GND | 共地 |
| DT（Data Out） | GPIO 26 | 序列資料輸出 |
| SCK（Serial Clock） | GPIO 25 | 時脈輸入，由 ESP32 控制 |

> **GPIO 26 / 25 為 `ScaleManager.h` 中的硬編碼預設值**，若需更換腳位須修改 `include/ScaleManager.h` 中的常數定義。

### 接線示意圖

```
NodeMCU-32S          HX711
──────────           ──────────
   3.3V  ───────────  VCC
    GND  ───────────  GND
  GPIO26 ───────────  DT  (Data)
  GPIO25 ───────────  SCK (Clock)
```

### 注意事項

1. **訊號線長度**：DT / SCK 線建議不超過 30 cm，過長容易受雜訊干擾
2. **電源去耦合**：若讀值不穩，可在 HX711 VCC 與 GND 之間接 100 µF 電解電容
3. **GPIO 共用**：確認 GPIO 26 / 25 未被其他模組（SPI、I2C、UART 等）占用

## Load Cell 惠斯通電橋接法

### 原理說明

四顆 50 kg 應變計以**惠斯通電橋（Wheatstone Bridge）**方式組合，可將量程提升至 200 kg，並自動消除溫度漂移與共模雜訊。電橋輸出為差動電壓（mV 等級），送入 HX711 進行放大與 ADC 轉換。

```
地墊俯視圖：

  LC1 ──────── LC2
  │              │
  │   地墊面積   │
  │              │
  LC4 ──────── LC3
```

四顆感測器放置於四角，承重時各自變形產生電阻變化。

### 接線圖

```
HX711
┌─────────────────────────────┐
│  E+  ─── LC1(紅) + LC3(紅)  │
│  E-  ─── LC2(黑) + LC4(黑)  │
│  A+  ─── LC1(白) + LC2(白)  │
│  A-  ─── LC3(白) + LC4(白)  │
└─────────────────────────────┘
```

> 不同廠商的 Load Cell 線色可能不同，請以實際規格書確認。常見顏色定義：
> - 紅線：激勵正（E+）
> - 黑線：激勵負（E-）
> - 白線：訊號（A+ 或 A-）
> - 綠線：訊號（A+ 或 A-）

### 並聯方式（四分之一橋並聯為全橋）

若採用四顆單獨的 50 kg Load Cell（非已配對全橋），接法如下：

```
           E+（HX711）
            │
   ┌────────┤────────┐
   │        │        │
  LC1      LC2      共用激勵
   │        │
   A+(HX711) A-(HX711)

四顆全部並接至同一對 E+/E-；
A+ 接 LC1 + LC3 的訊號正端；
A- 接 LC2 + LC4 的訊號正端。
```

## 地墊機構建議

### 感測器安裝

- 感測器固定於地墊底板（木板或壓克力板）四角，距邊緣約 5–10 cm
- 感測器上方加裝厚度一致的墊片（上板），確保踩踏時力量均勻傳遞至感測器
- 建議使用 M4 螺絲 + 螺母固定，避免感測器滑動

### 接地防護

- 若環境靜電明顯（地毯、冬季），可在地墊底板與 ESP32 GND 之間加接 1 MΩ 洩放電阻
- HX711 模組板上通常有 ESD 保護，無需額外處理

### 承重限制

- 四顆 50 kg = 總量程 200 kg（理論值）
- 建議實際使用不超過 150 kg，留有安全裕量
- 切勿施加衝擊力（如跳躍），應變計可能因瞬間過載而損壞
