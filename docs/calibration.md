# 秤的校正

## 為何需要校正

HX711 是 24-bit ADC，輸出的是原始數位值（Raw Value），與實際重量的關係為：

```
重量 (kg) = Raw Value ÷ Calibration Factor
```

**Calibration Factor** 因以下因素而每套硬體不同：
- Load Cell 靈敏度規格誤差（批次差異）
- 四顆 Load Cell 的並聯電阻匹配程度
- HX711 模組的增益設定（預設 128，可改 64 或 32）
- 接線長度和電阻影響

因此，**預設係數 2280.0 幾乎不可能準確**，首次使用前必須重新校正。

---

## 校正前置條件

1. 硬體已正確接線並上電（參考 [hardware.md](hardware.md)）
2. 裝置已連上 WiFi 並可存取 Web 設定頁
3. 地墊放置於**水平且穩定的地面**（傾斜會造成誤差）
4. 準備一個**已知精確重量的物體**（建議 > 20 kg，越重越準）
   - 最佳選擇：自己站上去（需知道體重）
   - 或：裝滿水的容器（1 公升水 ≈ 1 kg，可精確計算）

---

## 方法一：API 自動校正（推薦）

這是最簡單且最準確的方法。

### 步驟

1. **空秤確認**：確認地墊上沒有任何重物（人、物品）
2. **重啟裝置**（確保零點基準正確）：
   - 斷電再上電，或
   - Serial 監控中確認 `[Scale] tare OK`
3. **站上地墊**（或放置已知重量物體）並**保持靜止**
4. **等待 3–5 秒**讓讀值穩定
5. **呼叫校正 API**（在瀏覽器網址列或 curl）：
   ```
   http://<裝置IP>/api/calibrate?w=<你的實際體重kg>
   ```
   例如：`http://192.168.1.100/api/calibrate?w=72.5`
6. **記下 factor 值**：
   ```json
   { "ok": true, "raw": 165420, "factor": 2281.66 }
   ```
7. **更新設定頁**：
   - 開啟 `http://<裝置IP>/`
   - 在「秤參數」→「Calibration Factor」填入 `2281.66`（你得到的值）
   - 點選「儲存設定」，裝置重啟
8. **驗證**：再次站上並觀察 Serial 輸出的 `[Weight]` 值是否接近你的體重

### 常見 API 錯誤

| 錯誤訊息 | 原因 | 解決方法 |
|---------|------|---------|
| `missing param w` | 未傳入 `w` 參數 | 網址加上 `?w=<重量>` |
| `weight must be >= 5 kg` | `w` 值小於 5 | 傳入至少 5 kg 的值 |
| `no weight detected` | 地墊沒有放重物或讀值太低 | 確認有站上地墊且校正係數不是 0 |
| `wiring reversed? raw=...` | 原始讀值為負數 | 檢查 Load Cell A+/A- 是否接反 |
| `scale not initialized` | HX711 未就緒 | 重啟裝置後再試 |

## 方法二：手動兩點校正

適合想理解校正原理或 API 無法使用時。

### 步驟

1. **空秤，重啟裝置**，等待 Serial 顯示 `[Scale] tare OK`
2. **記錄零點**：空秤時原始讀值應接近 0（tare 已歸零）
3. **放置已知重量** W（kg）並等待 3 秒
4. **讀取原始值**：在瀏覽器開啟校正 API，讀取 `raw` 值：
   ```
   http://<IP>/api/calibrate?w=999  (w 值此時不重要，只看 raw)
   ```
5. **計算係數**：
   ```
   Calibration Factor = raw ÷ W
   ```
   例如：raw = 165420，W = 72.5 kg → Factor = 165420 ÷ 72.5 = 2281.66
6. **更新設定頁**並儲存

---

## 多點校正（進階）

若需要更高精度（例如同時使用的體重範圍很寬），可以進行多點校正：

1. 用三個不同重量（如 20 kg、50 kg、80 kg）分別量測 raw 值
2. 用線性回歸計算最佳係數：
   ```
   Factor = Σ(raw × weight) / Σ(weight²)
   ```
3. 三點校正通常能將誤差控制在 ±0.5 kg 以內

---

## 校正後驗證

校正完成後，使用已知重量物體驗證精度：

1. 放置不同重量物體（如 10 kg、30 kg、70 kg）
2. 觀察 Serial 輸出的 `[Weight] x.xx kg`
3. 計算誤差 = |測量值 - 實際值|

**可接受誤差**：
- 在 `matchToleranceKg` 範圍內（預設 3 kg）即可正常識別使用者
- 若需更精確，可縮小容差或重新進行多點校正

## 常見問題

### 讀值持續跳動（不穩定）

**原因**：
- 地墊放置不平
- 接線鬆動或過長（> 30 cm）
- 電源供電不穩（USB 線品質差）
- HX711 模組附近有強磁場干擾

**解決**：
- 增加取樣次數（修改 `main.cpp` 中的 `readWeightKg(3)` → `readWeightKg(10)`）
- 使用短且品質好的 USB 線
- 在 HX711 VCC/GND 間加 100 µF 電解電容

### 校正後仍不準確

**可能原因**：
- 校正時地墊不夠水平
- 四顆 Load Cell 靈敏度差異過大（批次問題）
- 校正重量本身不準確

**建議**：
- 用三種不同重量驗證，觀察誤差是否隨重量線性增長（代表係數本身有問題）還是固定誤差（代表零點偏移）

### 讀值為負數

HX711 A+/A- 訊號線接反，交換即可。
